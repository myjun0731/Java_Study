# Java-Ver.3

자바 스터디

## 예제 코드 구성

`examples` 각 챕터별 예제 코드입니다.

- `examples/Part1_Basics`: 자바 기초
- `examples/Part2_OOP`: 객체 지향 프로그래밍
- `examples/Part3_Essential_API`: 자바 핵심 라이브러리
- `examples/Part4_Data_Structure`: 컬렉션 프레임워크 & 제네릭
- `examples/Part5_Advanced_Modern_Java`: 모던 자바 & 심화

---

### **Part 1. 자바 기초 (The Basics)**

프로그래밍의 기본 흐름과 문법을 익히는 단계입니다.

- **Chapter_1 : Hello World & Main** (기본 구조, 출력문)
- **Chapter_2 : Variables & Types** (변수, 기본형 데이터 타입, 형변환)
- **Chapter_3 : Operators** (산술, 비교, 논리, 삼항 연산자)
- **Chapter_4 : Conditionals_IF** (if, else if, else)
- **Chapter_5 : Conditionals_Switch** (switch, case, 자바 12+ switch expression)
- **Chapter_6 : Loops_For** (기본 for문, 향상된 for문)
- **Chapter_7 : Loops_While** (while, do-while, break, continue)
- **Chapter_8 : Arrays_1D** (1차원 배열 선언, 초기화, 순회)
- **Chapter_9 : Arrays_Multi** (2차원 배열 및 다차원 배열)
- **Chapter_10 : Methods** (메서드 선언, 매개변수, 반환값, 오버로딩)

### **Part 2. 객체 지향 프로그래밍 (OOP)**

자바의 핵심인 객체 지향 개념을 다룹니다.

- **Chapter_11 : Class & Object** (클래스 정의, 객체 생성, 인스턴스)
- **Chapter_12 : Constructors** (생성자, this, 기본 생성자)
- **Chapter_13 : Variables_Scope** (멤버 변수, 지역 변수, static 변수)
- **Chapter_14 : Packages & Access Modifiers** (접근 제어자, import)
- **Chapter_15 : Encapsulation** (Getter, Setter, 데이터 보호)
- **Chapter_16 : Inheritance** (상속, extends, super)
- **Chapter_17 : Polymorphism** (다형성, 참조 변수의 형변환, instanceof)
- **Chapter_18 : Overriding** (메서드 오버라이딩 vs 오버로딩)
- **Chapter_19 : Abstract Class** (추상 클래스, 추상 메서드)
- **Chapter_20 : Interface** (인터페이스, implements, 다중 상속 효과)

### **Part 3. 자바 핵심 라이브러리 (Essential API)**

실무에서 가장 많이 쓰이는 기본 기능을 학습합니다.

- **Chapter_21 : Object Class** (equals, hashCode, toString)
- **Chapter_22 : String Class** (String 불변성, 주요 메서드)
- **Chapter_23 : StringBuilder & Buffer** (가변 문자열 처리)
- **Chapter_24 : Wrapper Classes** (Integer, Double, Auto-boxing)
- **Chapter_25 : Exception Handling** (try-catch-finally, throw, throws)
- **Chapter_26 : Custom Exceptions** (사용자 정의 예외 만들기)
- **Chapter_27 : Date & Time** (LocalDate, LocalDateTime, DateTimeFormatter)
- **Chapter_28 : Enums** (열거형 타입, 주요 메서드)

### **Part 4. 컬렉션 프레임워크 & 제네릭 (Data Structure)**

데이터를 효율적으로 관리하고 처리하는 방법을 배웁니다.

- **Chapter_29 : Generics** (제네릭 클래스, 메서드, 와일드카드)
- **Chapter_30 : Collection_List** (ArrayList, LinkedList)
- **Chapter_31 : Collection_Set** (HashSet, TreeSet, 중복 제거)
- **Chapter_32 : Collection_Map** (HashMap, TreeMap, Key-Value 구조)
- **Chapter_33 : Iterator & Comparator** (순회, 정렬 기준 정의)

### **Part 5. 모던 자바 & 심화 (Advanced & Modern Java)**

자바 8 이후의 기능과 고급 기법을 다룹니다.

- **Chapter_34 : Inner Classes** (내부 클래스, 익명 클래스)
- **Chapter_35 : Lambda Expressions** (람다식, 함수형 인터페이스)
- **Chapter_36 : Stream API_Basics** (스트림 생성, 필터링, 매핑)
- **Chapter_37 : Stream API_Advanced** (집계, 수집, 그룹화)
- **Chapter_38 : Optional** (NullPointException 방지)
- **Chapter_39 : I/O & Files** (입출력 스트림, 파일 읽기/쓰기)
- **Chapter_40 : Threads & Concurrency** (쓰레드 생성, 동기화, Runnable)
